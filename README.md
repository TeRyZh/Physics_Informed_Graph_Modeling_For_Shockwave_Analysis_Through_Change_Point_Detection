## Physics Informed Graph Modeling for Shockwave Analysis Through Change Point Detection

This repository implements a novel framework that integrates Physics-Informed Neural Networks (PINNs) with Graph Neural Networks (GNNs) to analyze and predict shockwave propagation. By leveraging Change Point Detection (CPD), the model accurately identifies the boundaries and transitions of shockwaves within graph-structured data, such as traffic road networks or fluid pipelines.

🌟 Key Features

1. Graph-Based Modeling: Captures spatial dependencies and complex network propagation dynamics using advanced Graph Convolutional layers.

2. Change Point Detection: Integrated statistical methods to detect abrupt transitions in data flow, marking the exact onset of shockwaves.

3. Spatio-Temporal Analysis: Designed to handle dynamic data across both spatial nodes and temporal sequences.
